QWERTY OS

A browser-based operating system simulator built with HTML, CSS, and JavaScript.

QWERTY OS recreates the feel of a desktop operating system inside a web browser, featuring movable windows, desktop applications, a taskbar, a start menu, persistent storage, and a modular kernel-inspired architecture.

⸻

Features

Desktop Environment

* Desktop background
* Draggable application windows
* Start menu
* Taskbar
* Window focus system
* XP-inspired user interface

Applications

* 🍋 Mango Engine
* 📁 File Explorer
* ⌨ Terminal
* ⚙ Settings
* 🧠 Task Manager
